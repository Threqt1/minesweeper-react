## Installation

1. Install all required dependencies using

```shell
npm install
```

or

```shell
pnpm add
```

or whatever package manager you're using

## Usage

1. Navigate to the main directory, and run

```shell
npm run dev
```

or

```shell
pnpm run dev
```

2. Navigate to the URL Vite provides in the console
